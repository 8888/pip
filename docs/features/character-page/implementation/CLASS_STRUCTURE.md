# Class Structure: Character Page

## DOM Structure

### `body`
- `bg-void`: Background color `#0d0208`.
- `text-bone`: Text color `#fbe9d0`.
- `font-mono`: Default font `Space Mono`.

### `.container`
- Centered layout with max-width.

### `#mascot-container`
- `pre` tag for Pip.
- Color: `#58a6ff`.

### `.stat-card`
- `bg-crypt`: `#1a0b14`.
- `border-2 border-gloom`.
- `p-6`.

### `.stat-row`
- `flex justify-between`.
- `.label`: Secondary text (`ash`).
- `.value`: Accent text (`toxic`).

### `.btn-radical`
- `bg-radical`: `#ff0055`.
- `text-void`: `#0d0208`.
- `border-0`.
- `uppercase font-bold`.
- Hard shadow: `4px 4px 0px 0px #0d0208`.
