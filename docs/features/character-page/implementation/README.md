# Feature: Character Page Implementation

This feature adds a new character profile page to the Pip habit tracker, following the "Cryptid Console" design system.

## Objective
Provide a dedicated space for users to view their character attributes and statistics.

## Components
1. **Character Header:** High-impact pixelated heading.
2. **Mascot Display:** Animated (static for now) ASCII Pip.
3. **Attribute Panel:** Grid/List of character stats.
4. **Retro Navigation:** Styled buttons for returning to the main dashboard.

## Style Guide Reference
All styles are derived from the `styleguide/` symlink, specifically the `Cryptid Console` toml configuration.
- **Void (#0d0208)**: Primary background.
- **Bone (#fbe9d0)**: Primary text.
- **Radical (#ff0055)**: Primary buttons.
- **Toxic (#00ff41)**: Stats/Success indicators.
